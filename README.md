Python Mini Systems using OOP
Project Overview

This repository contains multiple Python mini projects implemented using Object-Oriented Programming (OOP).
Each system simulates a real-world application such as parking management, billing systems, password managers, and more.

The project demonstrates the use of:

Classes

Objects

Inheritance

Dictionaries and lists

File handling

Loops and conditional statements

All systems run in the command line interface (CLI) and take input from the user.

Systems Included
1. Parking Lot Management System

Tracks vehicle entry and exit and calculates parking fees.

Features

Record vehicle entry time

Calculate parking fee based on hours

Track available parking spots

Manage vehicle exit

2. To-Do List Manager

A simple task manager that organizes tasks with categories and priorities.

Features

Add new tasks

Assign category and project tag

Set deadlines

Assign task priority

Display all tasks

3. Retail Store Billing System

Simulates a billing system used in retail stores.

Features

Add products with price

Scan products and quantity

Apply discounts

Generate bill

Record transactions

4. Movie Ticket Booking System

A basic movie booking system where users can book seats.

Features

Show movie details

Display available seats

Book seats

Track booked seats

5. College Result Management System

Manages student marks and calculates GPA.

Features

Store student details

Enter subject marks

Calculate total marks

Calculate GPA

Display final result

6. Password Manager

A simple password manager that stores encrypted passwords.

Features

Add new passwords

View saved passwords

Edit passwords

Password encryption using SHA-256

Save data in a JSON file

7. Freelancer Marketplace Simulation

Simulates a freelancing platform.

Features

Register freelancers

Register clients

Assign projects

Process payments

8. Warehouse Automation System

Manages inventory and product stock.

Features

Add products to inventory

Remove products

Generate inventory reports

Calculate total stock value

9. Customer Relationship Management (CRM)

Manages customer information and sales tracking.

Features

Add customer details

Record communication logs

Track sales pipeline

View customer data

Technologies Used

Python

Object-Oriented Programming (OOP)

JSON File Handling

Hashlib (for password encryption)

How to Run the Project

Clone the repository

git clone https://github.com/yourusername/python-mini-systems.git

Navigate to the folder

cd python-mini-systems

Run the program

python tasks.py
